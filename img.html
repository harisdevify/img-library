Three ways: cover, contain, fill

Same image, three different ways
object-fit: cover
object-fit: contain
object-fit: fill

</div>

Small file on mobile, big file on desktop - automatically

Browser automatically loads the right size

    small.jpg   400w,
    medium.jpg  800w,

08

Lazy Loading - Load Images When Needed

Image loads only when user scrolls to it - makes page faster

When to use: Always use this - put "eager" only on the first image, use "lazy" for everything else.
Blur effect that goes away when loaded
HTML + CSS 
<img
  src="image.jpg"
  alt="Image description"
  loading="lazy"
  class="blur-up"
>

/* CSS - Blur while loading */
.blur-up {
  filter: blur(12px);
  transition: filter 0.6s ease;
}
.blur-up.loaded {
  filter: blur(0);
}

// JavaScript
document.querySelectorAll('.blur-up').forEach(img => {
  if (img.complete) {
    img.classList.add('loaded');
  } else {
    img.addEventListener('load', () =>
      img.classList.add('loaded')
    );
  }
});

Skeleton Loader - Loading Animation

Show a shimmer effect while image loads

When to use: Slow internet users - instead of blank box, show a moving shimmer animation.
Watch the shimmer effect
HTML + CSS + JS 
<div class="skeleton-wrap">
  <img src="image.jpg" alt="..." class="skel-img">
</div>

/* CSS */
.skeleton-wrap {
  position: relative;
  background: #1c2840;
  border-radius: 10px;
  overflow: hidden;
  aspect-ratio: 16/9;
}
.skeleton-wrap::after {
  content: '';
  position: absolute;
  inset: 0;
  background: linear-gradient(90deg, transparent, rgba(255,255,255,0.06), transparent);
  animation: shimmer 1.6s infinite;
}
.skel-img {
  opacity: 0;
  transition: opacity 0.4s;
}
.skel-img.loaded { opacity: 1; }

Masonry Layout - Pinterest Style

Different height images together - CSS columns do it

When to use: Photo galleries, blog thumbnails - when images have different sizes.
3 column masonry - resize browser to see it change
HTML + CSS 
<div class="masonry">
  <div class="item">
    <img src="img.jpg" alt="..." loading="lazy">
  </div>
</div>

/* CSS - CSS columns trick */
.masonry {
  columns: 3;
  column-gap: 1rem;
}

/* Mobile - 2 columns */
@media (max-width: 600px) {
  .masonry { columns: 2; }
}

.item {
  break-inside: avoid;
  margin-bottom: 1rem;
  border-radius: 10px;
  overflow: hidden;
}

.item img {
  width: 100%;
  height: auto;
  display: block;
}

Accessibility - For All Users

Alt text, keyboard focus, reduced motion - do this always

When to use: Always - this is not just rules, real people depend on this.
HTML + CSS 
<!-- 1. Important image - write good alt text -->
<img src="product.jpg" alt="Blue leather bag">

<!-- 2. Decoration only - leave alt empty -->
<img src="divider.png" alt="">

<!-- 3. Complex image - use figure and figcaption -->
<figure>
  <img src="chart.jpg" alt="Sales chart">
  <figcaption>2024 Sales growth</figcaption>
</figure>

/* CSS - Keyboard focus visible */
img:focus-visible {
  outline: 3px solid #38bdf8;
  outline-offset: 4px;
  border-radius: 4px;
}

/* Respect user's motion preference */
@media (prefers-reduced-motion: reduce) {
  * {
    animation-duration: 0.01ms !important;
    transition-duration: 0.01ms !important;
  }
}

/* Print styles */
@media print {
  img {
    max-width: 100% !important;
    page-break-inside: avoid;
  }
}
